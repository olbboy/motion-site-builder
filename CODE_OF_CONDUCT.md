# Code of Conduct

This project follows the [Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

**In short:** be respectful, be constructive, assume good faith. Harassment, personal attacks, and gatekeeping are not tolerated anywhere in the project — issues, PRs, discussions, or elsewhere.

Design taste is subjective; critique the work with specifics ("this easing overshoots on mobile"), never the person.

Report unacceptable behavior by opening a confidential issue or contacting the maintainers directly. Maintainers may edit, remove, or reject contributions and contributors that violate this code.
